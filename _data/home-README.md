# Homepage Editing

Edit `/_data/home.yml` to update homepage content without touching template code.

- `carousel.slides`: hero images and captions.
- `research_recent.limit`: number of recent publications shown.
- `research_recent.pinned_publications`: optional list of publication IDs to pin specific items.
  Use the `publication_id:` value from each publication file.
- `welcome.*`: welcome card title/image/text.
- `news.title` and `news.limit`: right card heading and number of news items.
